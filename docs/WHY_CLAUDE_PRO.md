# Claude Pro Efficiency

For many individual developers and researchers, Claude Pro provides access to world-class reasoning with a generous message limit. 

### The Efficiency Gap
Users may find themselves hitting message limits before completing their tasks, often due to context management challenges:
- **Redundant Thread History:** Replying to long threads instead of starting clean sessions.
- **Context Bloat:** Including entire files when only small portions are relevant.
- **Output Repetition:** Requesting full file rewrites for minor changes.

### Our Focus
`claude-mileage` provides tools to reduce waste within your existing Claude CLI workflow. By focusing on compact context and local memory, you can more effectively complete complex tasks within your existing message limits. Our goal is to make every interaction as high-signal as possible.
